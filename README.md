# tiny-clock

> Just a tiny floating clock for macOS.

![demo](tiny-clock.gif)

## Getting started

Clone the repo.

```bash
git clone https://github.com/stursby/tiny-clock.git
```

Install dependencies

```bash
cd tiny clock && yarn
```

### Dev

Run the Electron app in Development

```bash
yarn start
```

Build the app for macOS

### Build

```bash
yarn build
```
